# PrintForge Ops starter package

Included:
- 16 .toon / markdown planning files
- 1 startup prompt for GitHub agents

Prompt length:
- 59991 characters

Use:
1. Drop the `.toon` files into your repo or docs folder.
2. Feed `startup_prompt_60000.md` to your coding agent as the main bootstrap prompt.
3. Use the `.toon` files as the persistent domain and architecture contracts.

Notes:
- The package is intentionally extensible and seed-based.
- It supports the requested printer fleet, Firebase auth/data layer, .3mf management workflow, filament taxonomy, and Siraya Tech resin catalog scaffolding.
